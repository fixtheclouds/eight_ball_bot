# eight_ball_bot

Test project for Crystal lang & Tourmaline framework

## Installation

```crystal
shards install
crystal run src/eight_ball_bot.cr
```

## Usage

Find [@EightBallCrystalBot](https://t.me/crystal_pet_bot) on telegram

Bot commands:
- `/shake` - print out an answer
